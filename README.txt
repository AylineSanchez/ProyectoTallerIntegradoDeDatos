# Proyecto de Análisis de Datos Fisiológicos

Este proyecto realiza análisis exploratorio de datos y modelado predictivo utilizando datos fisiológicos y calificaciones de estudiantes.

## Requisitos

* Python 3.7 o superior
* Bibliotecas: pandas, numpy, matplotlib, seaborn, sklearn, tkinter

## Instalación

1. Clona el repositorio:
 
git clone <URL_DEL_REPOSITORIO>

2. Instala las bibliotecas necesarias
 
Estructura del proyecto

Data/: Contiene los datos fisiológicos de los estudiantes (S1 a S10) y sus calificaciones (StudentGrades.txt).
ProyectoTaller.py: Script principal de Python con la interfaz gráfica y las funciones de análisis.

3. Ejecución

python ProyectoTaller.py

Uso

1. Al ejecutar el script, se abrirá una ventana con diferentes opciones de análisis.
2. Selecciona la opción deseada y sigue las instrucciones en pantalla.
3. Los resultados se mostrarán en forma de gráficos o tablas.

Opciones de análisis

Tendencias por Participante: Visualiza las tendencias de las variables fisiológicas a lo largo del tiempo.
Datos Contextuales - Rendimiento Académico: Explora la relación entre las variables fisiológicas y las calificaciones.
Boxplots de Variables Fisiológicas: Genera boxplots para comparar la distribución de las variables.
Técnicas Estadísticas para Resumir Hallazgos: Calcula estadísticas descriptivas de las variables.
Detección de Valores Atípicos: Identifica valores atípicos en las variables fisiológicas.
Modelado Predictivo: Crea modelos para predecir las calificaciones en función de las variables fisiológicas.
